## 📖 About

**Realistic Weapon Wheel** gives the default GTA V weapon wheel and in-vehicle radio interface a refreshed visual appearance.

The resource replaces the standard interface styling with a more realistic and colorful design, helping the weapon wheel feel less dated while keeping the familiar functionality players already know.

No complicated configuration. No unnecessary dependencies. Simply install the resource, start it, and you're ready to go.

## 📥 Installation

### 1. Download the Resource

Download the latest version of the resource and extract the downloaded ZIP file.

### 2. Add it to Your Server

Move the `WeaponWheel` folder into your FiveM server's resources directory.

For example:

    resources/[standalone]/WeaponWheel

### 3. Start the Resource

Add the following line to your `server.cfg`:

    ensure WeaponWheel

### 4. Restart Your Server

Restart your FiveM server or start the resource manually.

Once loaded, the updated weapon wheel and radio interface should automatically be visible in-game.
